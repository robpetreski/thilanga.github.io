1. Create a Bundle
php app/console generate:bundle --namespace=Acme/SearchBundle --format=yml

2.Test Created Bundle
vim app/config/routing.yml
Replace  prefix:   / with prefix:   /search
http://localhost/symfony/search/hello/thilanga

3. Create Search Service
mkdir -p src/Acme/SearchBundle/Service/
vim src/Acme/SearchBundle/Service/GlobalSearch.php
